# HAAMSTER
**A Silly Slack Bot For Daily Report**

*※ 주의: 욕하지 마시오.*

```
~/.bash_profile

export HAAMSTER_SLACK_TOKEN=<MY_SLACK_TOKEN>
export HAAMSTER_FIREBASE_URL=<MY_FIREBASE_URL>
```

## Run

```bash
$ npm start
```

## Development

### Auto restart on file change

```bash
$ npm run dev
```

### Test

```bash
$ npm test
```
